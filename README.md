Ejercicio 2, archivo BC3 II, REALIZADO EN CLASE Enunciado;2. Sets: Set de emails únicos
Crea un programa que gestione un conjunto de correos electrónicos registrados (HashSet<String>).
El programa debe permitir:
• 1: Registrar un email nuevo. Si ya existe, informar “Email ya registrado”.
• 2: Mostrar todos los emails registrados.
• 3: Comprobar si un email está registrado (contains).
• 0: Salir.
